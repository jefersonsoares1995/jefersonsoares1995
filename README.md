## 🚀 Projetos Alpha (Case Studies)

### 🧩 Agentes AI - Hub de Produtividade
Plataforma que unifica agentes inteligentes para otimizar tarefas de squads técnicos e criativos.
* **Arquitetura:** `Lovable` (Frontend) + `Supabase` (Backend) + `n8n` (Orquestração) + `OpenAI` (Inteligência).
* **Impacto:** Centralização de processos de IA em um único ecossistema personalizado para definições técnicas e marketing.

### 🧠 HELP IA (Inteligência de Atendimento)
Sistema de suporte automatizado que utiliza LLMs para resolver chamados complexos com contexto real de negócio.
* **Tecnologias:** `n8n`, `OpenAI`, `Python`, `Airtable`.

### ⚡ Automação de Workflow Financeiro
Pipeline de RPA para conciliação bancária e processamento de guias fiscais (GNRE) na indústria.
* **Tecnologias:** `Python` + `n8n` + `Integração ERP`.
